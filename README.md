# WeatherApp
A simple Weather App built with HTML, CSS, and JavaScript that fetches real-time weather data using the OpenWeatherMap API. Users can search any city to view its temperature, humidity, and wind speed.  Tech: HTML • CSS • JavaScript • OpenWeatherMap API
